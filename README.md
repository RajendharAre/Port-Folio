## üö¶ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/RajendharAre/Port-Folio.git
cd Port-Folio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the server

```bash
npm start
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.
